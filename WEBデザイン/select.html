SELECT

手帳の種類については、以下のように選ぶことができます:

  1. 　多くの手帳ブランドでは、時間をかけて計画を立てたり、日々の生活を記録してストックする必要がある人のために、ウィークリープランナーやマンスリープランナーを用意しているのが一般的です。 これらの本には、1年・1ヶ月・1日の計画ページだけでなく、自由に遊べる日付のない空白のページもあるのが普通です。
  2. 　「自分の人生にはそんなに記録したり計画したりすることはないし、毎日書くのは面倒だし、続けるのは難しいのではないかと心配」という方は、日付の入っていない手帳や自分で記入するタイプの日付手帳を使ってみてください。 いつでも好きなだけ書けるし、記入しないと格好がつかないという心理的負担もないので、初心者が始めるには最適です。
  3. 　勉強の記録や観劇の記録、台帳の作成など、特定の目的がある場合は、専用の手帳を選ぶと、自分で図を描いたり、表を作ったり、フォーマットをデザインしなくても、すぐに使えてとても便利です。
  4. 　多くの手帳ブランドでは、定規やシール、テープ、スタンプなど様々なツールや周辺機器を提供する。 手帳をつける習慣がつくまでは買わずに、手帳を飾るために使うのがおすすめです。 道具や周辺機器はあくまでもおまけで、ペンで何を書くかによって、あなたの手帳は変わってくるのです。
